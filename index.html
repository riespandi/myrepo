How to use Debian or Ubuntu Repository ?

Run these commands to add this repostory to you APT source list:

Ubuntu:

$ sudo echo "deb http://repo.freeweb.id/ubuntu <version> main" > /etc/apt/sources.list.d/freeweb.list

Debian:

$ sudo echo "deb http://repo.freeweb.id/debian <version> main" > /etc/apt/sources.list.d/freeweb.list

Add our public key then update the packages list:

$ wget -qO - http://repo.freeweb.id/pubkey.asc | sudo apt-key
$ sudo apt update

How to use Debian or F-Droid Repository ?

  1. Install the F-Droid Client app. Get it here »
  2. Run the app and go to Menu > Repositories
  3. Add new repo: https://repo.freeweb.id/apk
  4. Turn on the Ertix Android then refresh repository
